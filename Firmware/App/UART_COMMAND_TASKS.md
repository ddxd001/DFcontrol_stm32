# UART 指令与任务对应文档

本文档对应当前 `Firmware/App/app_tasks.c` 的实现，用于快速查看 `USART1` 指令与任务/动作映射关系。

## 1. 基本协议

- 帧格式：`AA BB XX`
- 入口：`App_Task_10ms()` 内串口状态机解析
- 串口：`USART1`（`DebugUart`）
- 上电保护：前约 `1200ms` 会忽略触发请求

## 2. 指令总表

| 指令 | 功能 |
|---|---|
| `AA BB 01` | 蜂鸣器触发一次 |
| `AA BB 02` | 触发 Q1 |
| `AA BB 03` | 触发 Q2-1 |
| `AA BB 04` | 触发 Q2-2 |
| `AA BB 05` | 触发 Q3-1 |
| `AA BB 06` | 触发 Q3-2 |
| `AA BB 07` | 触发 Q4 |
| `AA BB 08` | 步进电机使能 |
| `AA BB 09` | 步进电机失能 |
| `AA BB 0A` | 步进电机左转 90°（相对位移） |
| `AA BB 0B` | 步进电机右转 90°（相对位移） |
| `AA BB D0` | 测试任务 TEST1 |
| `AA BB D1` | 测试任务 TEST2 |

## 3. 任务动作定义

## Q1（`AA BB 02`）

- 先前进 `0.08m`
- 转向/直走循环执行（共 4 轮）
- 最后一轮直走使用 `0.92m`
- 结束停车并蜂鸣

## Q2-1（`AA BB 03`）

- 先前进 `0.08m`
- 左转 90° + 直走循环（3 轮）
- 调头并返程（两段 `1.0m` + 最后一段 `0.9m`）
- 结束停车并蜂鸣

## Q2-2（`AA BB 04`）

- 先前进 `0.08m`
- 右转 90° + 直走循环（3 轮）
- 调头并返程（两段 `1.0m` + 最后一段 `0.9m`）
- 结束停车并蜂鸣

## Q3-1（`AA BB 05`）

- 基本流程与 Q1 同级（总共 4 轮）
- 第 2 段直走已替换为 TEST1 的整段流程：
  - 匀速前进 `0.3m`
  - 蜂鸣一次
  - 匀速前进 `0.4m`（开启路口检测）
  - 蜂鸣连续两次
  - 自适应前进 `0.3m`
- 第 2 段执行完成后回到 Q3-1 原循环逻辑继续后续轮次
- 路口检测频率：`100Hz`（10ms）
- 每次检测到新路口：
  - 蜂鸣一次
  - UART1 发送：`CROSS:<n>`
  - UART1 发送相邻路口距离：`DIST:xx.xcm`

## Q3-2（`AA BB 06`）

- 直接匀速前进 `1.0m`（速度 10）
- 等待后结束停车并蜂鸣

## Q4（`AA BB 07`）

- 前进 `0.08m`
- 左转 `90°`
- 前进 `0.55m`
- 等待 `4.0s`
- 前进 `0.45m`
- 左转 `90°`
- 前进 `0.92m`
- 结束停车并蜂鸣

## TEST1（`AA BB D0`）

- 匀速前进 `0.3m`
- 蜂鸣一次
- 匀速前进 `0.4m`（开启路口检测）
- 蜂鸣连续两次
- 自适应前进 `0.3m`
- 结束停车并蜂鸣

### 路口间距测量（用于 TEST1，且已复用到 Q3-1 第2段）

- 测量对象：相邻两次路口触发间距
- 速度常量：`0.33 m/s`
- 修正系数：`0.71`
- 输出单位：厘米，保留 1 位小数
- UART1 输出格式：`DIST:xx.xcm`

## TEST2（`AA BB D1`）

- 匀速前进 `1.0m`（速度 10）
- 稳定等待后停车并蜂鸣

## 4. 步进电机说明（`08/09/0A/0B`）

- 驱动对象：`StepperMotorDrv`
- 串口：`USART2`（当前工程引脚配置为 `PA2 TX / PD6 RX`）
- `0A/0B` 参数：
  - 位置模式：相对
  - 速度：`100`
  - 脉冲：`±800`（90° 估算值，需按实机减速比/细分标定）

## 5. 串口输出信息

- 路口事件：`CROSS:<n>`
- 路口间距：`DIST:xx.xcm`（TEST1 与 Q3-1 第2段）
